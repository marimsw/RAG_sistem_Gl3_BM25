Этот код реализует две версии алгоритма BM25 (Best Matching 25) для релевантного текстового поиска:

BM25Retriever — детальная реализация с возможностью объяснения скоринга

Индексирует документы, строит словарь терминов

Вычисляет статистики (IDF, частоты терминов, длины документов)

Оценивает релевантность запроса к документам по формуле BM25

Поддерживает объяснение вклада каждого термина в итоговый скор

OptimizedBM25 — оптимизированная версия для больших корпусов

Использует разреженные матрицы (scipy.sparse) для эффективного хранения

Векторизованные вычисления для быстрого поиска

Ускоренная обработка за счёт матричных операций

Основное назначение: Ранжирование документов по релевантности к текстовому запросу, используется в поисковых системах и RAG-системах как эффективная альтернатива TF-IDF.
